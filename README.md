Smart Leads Dashboard

A full-stack MERN + TypeScript Lead Management Dashboard.

Tech Stack :-

. Frontend: React, TypeScript, TailwindCSS, React Router, React Hook Form, Axios
. Backend: Node.js, Express, TypeScript, MongoDB, Mongoose, JWT
. DevOps: Docker, Docker Compose

Features :-

. JWT Authentication with bcrypt
. Role-Based Access Control (Admin / Sales)
. CRUD operations on Leads
. Advanced filtering, debounced search, sorting
. Server-side pagination
. CSV export
. Dark mode
. Responsive UI
. Loading, empty, and error states
. Centralized error handling
. Form validation (both client and server)

Setup (Local)

Prerequisites :-
. Node.js 18+
. MongoDB (local or Atlas)

Backend :-
```bash
cd backend
cp .env.example .env
npm install
npm run dev
